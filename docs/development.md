# Development

## Requirements

- **Go 1.21+** (for `lib-gen`)
- **Helm 3** (for chart development and testing)
- [`helm unittest`](https://github.com/helm-unittest/helm-unittest) plugin (for running chart tests)

---

## Repository Setup

```bash
git clone https://github.com/Brains-Beyond-Expectations/bbe-charts.git
cd bbe-charts

# Install the helm-unittest plugin
helm plugin install https://github.com/helm-unittest/helm-unittest
```

---

## Running Chart Tests

Each chart has a `tests/` directory with `helm-unittest` YAML test suites.

Run tests for a specific chart:

```bash
helm unittest charts/sonarr
helm unittest charts/bbe-networking
```

Run tests for all charts:

```bash
helm unittest charts/*
```

---

## Updating a Chart

1. Make changes to the chart under `charts/<name>/`
2. Bump the chart version in `charts/<name>/Chart.yaml`
3. Regenerate `library.yaml` (see below)
4. Run chart tests to verify nothing is broken

---

## Regenerating `library.yaml`

`library.yaml` is autogenerated and must never be edited by hand. After bumping a chart version, run `lib-gen`:

```bash
cd lib-gen
go run main.go
```

Or via the Makefile:

```bash
cd lib-gen
make generate
```

`lib-gen` reads `required-versions.yaml` and all `Chart.yaml` files, then rewrites `library.yaml` with the updated version history.

See [Library Versioning](library-versioning.md) for the full algorithm.

## Adding a New Chart

### Individual Chart

1. Create `charts/<name>/` with the standard chart structure:
   - `Chart.yaml`
   - `values.yaml`
   - `values.schema.json` (optional but recommended)
   - `templates/` — deployment, service, ingress, PVC templates
   - `tests/` — helm-unittest test suites
2. Add documentation at `docs/charts/<name>.md`

### Umbrella Chart

1. Follow the individual chart steps above
2. Add the chart as a dependency in the relevant umbrella chart's `Chart.yaml` (`bbe-media` or `bbe-networking`)
3. Run `helm dependency update charts/<umbrella-name>` to download the dependency
4. Add an entry to `required-versions.yaml` if this is a new top-level installable package
5. Run `lib-gen` to regenerate `library.yaml`

## Renovate

The repository includes a `renovate.json` configuration. Renovate Bot automatically opens pull requests to update container image tags in `Chart.yaml` and external Helm chart dependency versions.
